# Sierra (sierra-ai)

Sierra is a conversational AI platform for customer experience, founded in 2023 by Bret Taylor (ex-co-CEO of Salesforce, ex-CTO of Facebook, co-creator of Google Maps, and chair of OpenAI's board) and Clay Bavor (ex-Google VP for Labs, Project Starline, Lens, and Workspace). Headquartered in New York with offices in San Francisco, Atlanta, London, Paris, Madrid, Toronto, Singapore, and Tokyo. Sierra deploys branded AI agents for customer service, sales, and operations across chat, voice, SMS, WhatsApp, email, and ChatGPT for enterprises including SiriusXM, WeightWatchers, Sonos, ADT, Chime, Sutter Health, Rocket Mortgage, Ramp, Brex, SoFi, CLEAR, and roughly 40% of the Fortune 50.

## APIs

Sierra does not currently offer a public, self-serve developer API. The Agent SDK and the `docs.sierra.ai` developer documentation portal are gated and available only to contracted enterprise customers. There is no public GitHub organization, no published OpenAPI specification, no public SDK package, and no public sandbox or sign-up. Access is via enterprise sales and an outcome-based commercial contract.

This catalog entry therefore profiles Sierra as a closed-platform AI agent provider rather than generating placeholder OpenAPI, JSON Schema, capability, plan, rate-limit, or FinOps artifacts. If Sierra publishes a public API surface in the future, this repository will be updated to include machine-readable specifications.
